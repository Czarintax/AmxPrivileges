# AmxPrivileges
Публичная страница привилегий AMX Mod X по серверам: карточки с прогресс-баром или компактная таблица, модалка с подробностями. Аватарки тянутся из Steam, привязываются по SteamID; IP-адреса скрыты от чужих глаз.  В профиле игрока появляется вкладка с его привилегиями.
